## Como Usar

### Acessar o Dashboard
- **URL:** https://jarvis-nexus.up.railway.app
- **Login:** `jarvis2026`

### Chat Básico (Sem JARVIS)
Funciona direto no site, sem precisar ligar o JARVIS:
- Digite "oi" - Recebe saudação
- Digite "hora" - Mostra hora atual
- Digite "data" - Mostra data atual
- Digite "status do pc" - Mostra CPU/RAM do servidor

### Câmera
1. Vá na página "Visão"
2. Clique "Ativar Camera"
3. Permita o acesso à câmera
4. Pronto! Câmera funciona pelo navegador

### Controle de Voz
1. Vá na página "Voz"
2. Clique no botão do microfone
3. Fale algo
4. O site processa pelo navegador

## Configuração de IA (Opcional)

Para ter respostas inteligentes como o JARVIS original:

### Criar arquivo `.env` na raiz do projeto:
```env
# Provedor de IA (groq é gratuito)
AI_PROVIDER=groq
AI_MODEL=llama-3.3-70b-versatile

# Chave de API do Groq (gratuita)
GROQ_API_KEY=sua_chave_aqui

# Opcional: Gemini
# GEMINI_API_KEY=sua_chave_aqui
```

### Obter chave Groq (gratuita):
1. Acesse https://console.groq.com
2. Crie uma conta
3. Vá em "API Keys"
4. Crie uma chave
5. Copie e cole no arquivo `.env`

## Solução de Problemas

### Chat não responde
- Verifique se está no modo básico (sem IA)
- Digite "ajuda" para ver comandos disponíveis
- Se quiser IA, configure as API keys no `.env`

### Câmera não funciona
- Verifique se o navegador tem permissão
- Teste em outro navegador
- Verifique se o dispositivo tem câmera

### Login não funciona
- Senha padrão: `jarvis2026`
- Se esqueceu, delete o arquivo `data/auth_config.json`
- O sistema cria uma nova senha padrão

### Site muito lento
- Railway desliga após 15min sem uso
- Primeira requisição pode demorar ~30s
- Isso é normal no plano gratuito
